Git Flow

1. To initialize a git folder as local repo
    
    `git init`

2. To add files into git local repo
   
    `git add <file>`

3. To commit changed files to local repo

    `git commit -m "<message here>"`

4. To push to remote repo

    `git push`

5. To clone remote repo
    
    `git clone <url>`
